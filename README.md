## Instalação

1. Clone o repositório:

    ```bash
   git clone https://github.com/williansnex/todolist-angular.git

2. Navegue até o diretório do projeto:
   ```bash
   cd todolist-angular

3. Instale as dependências:
   ```bash
   npm install

4. Inicie o aplicativo:
   ```bash
   ng serve
